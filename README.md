# Walmart Retail Sales Analysis 🛒

## Overview
End-to-end analysis of $6.74B Walmart sales data 
across 45 stores and 3 years using Python, SQL, Excel & Power BI.

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (SQL Server)
- Microsoft Excel
- Power BI

## Key Findings
- August is peak sales month — 95% higher than January
- Type A stores contribute 64% of total revenue
- Holiday weeks drive higher average weekly sales
- Department 92 is top performing across all stores

## Dataset
Walmart Retail Dataset — Kaggle (445K+ rows, 3 CSV files)
